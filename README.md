## Descrição
Aplicação em Vue 3 + Vuetify 3 que permite criar, listar, editar e excluir receitas.  
Os dados persistem em `localStorage`.  
Possui 3 rotas: **Home**, **Receitas** e **Sobre**.

## Tecnologias
- Vue 3
- Vue Router 4
- Vuetify 3
- Vite
